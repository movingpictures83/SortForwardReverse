# SortForwardReverse
# Language: Python
# Input: TXT
# Output: DIR
# Tested with: PluMA 1.1, Python 3.6

Sort forward and reverse reads to be in the same order

The input parameter file contains a set of tab-delimited keyword-value pairs:

forwardPrefix: Prefix for input FASTA and QUAL forward read files, assumed prefix.fasta and prefix.qual
reversePrefix: Same, but for reverse reads
sortedForwardPrefix: Prefix for output FASTA and QUAL forward read files, assumed prefix.fasta and prefix.qual
sortedReversePrefix: Same, but for reverse reads

Output FASTA and QUAL files will be generated to the user-specified output directory
